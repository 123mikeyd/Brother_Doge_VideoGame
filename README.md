# HODL OR DIE: Brother Doge

A Dogecoin-themed HTML5 survival shooter where several flavors of Doge fight the Wicked Paper Hand mob.

## Play

Play in browser:

https://123mikeyd.github.io/Brother_Doge_VideoGame/

If GitHub Pages is not live yet, open `index.html` or `BrotherDoge.html` locally in a browser.

## What changed in this build

- Fixed health display rounding, so HP no longer shows floating-point junk like `220.00000000000003`.
- Added a Wave 1 training moment: kill one unmoving practice Paper Hand before the real run starts.
- Added in-game control explanations for keyboard, mouse, mobile touch, and gamepad.
- Added persistent progress with `localStorage` for unlocked weapons, highest difficulty, best run stats, and Degen Mode discovery.
- Added run summaries after death and victory.
- Added boss health bar and clearer boss danger telegraphs.
- Added `index.html` so the game is playable from GitHub Pages.

## Features

- 5 playable Doge breeds with different playstyles
- Wave-based survival shooter gameplay
- Shop between waves
- Weapons, upgrades, and 40+ perks
- Roguelite carryover after victory
- Difficulty mutations across repeated clears
- Boss fight against the Wicked Paper Hand
- Secret Degen Mode
- Browser-only: no install required

## Controls

Keyboard / mouse:

- WASD or Arrow Keys: move
- Mouse: aim
- Hold mouse button: shoot
- Esc: pause

Gamepad:

- Left stick: move
- Right stick: aim / shoot
- A or right trigger: shoot

Mobile / touch:

- Left side: virtual movement stick
- Right side: aim and shoot
- If using only one thumb, the game auto-aims toward nearby enemies while moving

## Run flow

1. Pick a Doge.
2. Kill the practice Paper Hand to prove you are ready.
3. Survive waves.
4. Buy weapons and perks in the shop.
5. Beat the boss.
6. Carry some progress into the next difficulty.
